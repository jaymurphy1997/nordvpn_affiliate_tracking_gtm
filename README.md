# Google Tag Manager script for NordVPN Affiliate Tracking
This is the script for an Custom HTML tag in Google Tag Manager.  It will append query string parameters to a NordVPN link which in turn will provide data that can be used for a Postback URL sneding Transaction information to Google Analytics 4 and additional data that can be used to send postback information to advertising outlets.  One example included is YEP advertsing data - click_id, pub_id and sub_id.
